# Agenda em Python

Este é um programa simples de agenda em Python que permite gerenciar contatos de forma interativa. Você pode adicionar, listar, buscar, editar e remover contatos da sua agenda.

## Funcionalidades

- Adicionar novos contatos
- Listar todos os contatos
- Buscar contato pelo nome
- Editar informações de contato
- Excluir contatos

## Como funciona

O programa mantém uma lista de contatos, onde cada contato é representado por um dicionário contendo nome, email e telefone. Através de um menu interativo, você pode realizar diversas operações para gerenciar seus contatos.

## Como usar

1. Clonar ou copiar o arquivo para sua máquina.
2. Executar o script Python.
3. Utilizar o menu para gerenciar seus contatos.# agendaDeContatosPython
